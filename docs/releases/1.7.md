# NativeGallery 1.7

**Дата:** 2026-07-02  
**Тег:** `release-1.7`  
**Релиз:** [GitHub Releases](https://github.com/Lanmikeman/nativegallery/releases/tag/release-1.7)  
**Репозиторий:** [Lanmikeman/nativegallery](https://github.com/Lanmikeman/nativegallery)

SPA-навигация без перезагрузки, перелистывание фото, улучшения админ-панели и исправления страницы фото.

## Основное

### SPA и фото

- `routing.js` — навигация без полной перезагрузки (для авторизованных).
- `GET /api/photo/move` — перелистывание фото стрелками.
- `ng_asset()` — сброс кэша JS/CSS по git-hash.
- Страница фото: inline-навигация `#prev` / `#next` и контекст `pid`/`vid`/`gid` для SPA.

### Админ-панель

- Сгруппированное боковое меню.
- Полный список фото с пагинацией и фильтрами.
- Редактор страниц через API.

### Исправления

- Перелистывание фото после SPA-переходов.
- Главная: кликабельные случайные фото, корректные URL `/photo/{id}`.
- SPA на `/photo/`: без дублирующего подвала, корректное размещение `#pmain`.
- Заголовок на `/page/{id}`, предупреждения PHP на странице комментариев и в профиле.
- Редактор страниц в админке (загрузка контента через API).

## Обновление с 1.6

```bash
cd /var/www/nativegallery   # или ваш NG_WEB_ROOT (пример: /mnt/win/nativegallery)
git fetch --tags
git checkout release-1.7    # или git pull origin main
composer install --no-dev

# опционально — страницы с датой правки (если пропустили 1.6+):
mysql -u USER -p DATABASE < sqlcore/sql_0009.sql

sudo chown -R www-data:www-data storage uploads cdn logs
sudo chmod -R 775 storage uploads cdn logs

sudo systemctl reload nginx php8.3-fpm
```

Обновите страницу в браузере с полным сбросом кэша (Ctrl+F5), чтобы подтянулись новые JS/CSS.

## Nginx

**Правки конфигурации nginx для релиза 1.7 не требуются**, если сайт уже работает через front controller (`deploy/nginx/nativegallery.conf`).
