# NativeGallery 1.6

**Дата:** 2026-07-02  
**Тег:** `release-1.6`  
**Релиз:** [GitHub Releases](https://github.com/Lanmikeman/nativegallery/releases/tag/release-1.6)  
**Репозиторий:** [Lanmikeman/nativegallery](https://github.com/Lanmikeman/nativegallery)

Актуальное состояние ветки `main`: OpenVK и админка из 1.5, фотоконкурсы, личный кабинет, роль владельца, страницы помощи и отчётов конкурса.

## Основное

- **OpenVK** — вход, привязка аккаунта, управление инстансами через overlay `storage/auth-settings.json`
- **Админка** — права пользователей, настройки авторизации, создание конкурсов, задачи cron
- **Роль владельца (`admin = 4`)** — переключатель Debug (Tracy), редактирование конфига сервера (`storage/server-settings.json`)
- **Фотоконкурсы** — голосование, претенденты, победители, рейтинг, `/pk.php` (отчёт по конкурсу)
- **Личный кабинет** — `/lk/ticket.php` (заявки БД), `/lk/konkurs.php` (участие в конкурсе)
- **Публичные страницы** — `/help/`, слоган подвала из `footerslogan` в конфиге
- **Редактирование новостей** — админка, пометка «отредактировано»

## Обновление с 1.4

```bash
git pull
composer install --no-dev

mysql -u USER -p DATABASE < sqlcore/sql_0008.sql

sudo chown -R www-data:www-data storage
sudo chmod -R 775 storage
sudo NG_WEB_ROOT=/mnt/win/nativegallery bash deploy/setup-cron.sh

sudo systemctl reload nginx php8.3-fpm
```

## Первый владелец сервера

```sql
UPDATE users SET admin = 4 WHERE id = ВАШ_ID;
```

Раздел **Админка → Сервер** появится только у пользователя с `admin = 4`.
