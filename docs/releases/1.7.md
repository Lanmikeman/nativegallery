# NativeGallery 1.7

**Дата:** 2026-07-02  
**Тег:** `release-1.7`  
**Релиз:** [GitHub Releases](https://github.com/Lanmikeman/nativegallery/releases/tag/release-1.7)  
**Репозиторий:** [Lanmikeman/nativegallery](https://github.com/Lanmikeman/nativegallery)

Музыкальный раздел, SPA-навигация без перезагрузки, общие радиостанции от администратора, исправления страницы фото и плеера.

## Основное

### Музыка (`/music`)

Доступно авторизованным пользователям.

| Возможность | Описание |
|-------------|----------|
| Загрузка треков | MP3, OGG, WAV, FLAC и др. (`POST /api/audio/upload`) |
| Личные потоки | HTTP-радио и прямые URL (`POST /api/audio/stream`, `/api/audio/url`) |
| M3U | Импорт и воспроизведение плейлистов в браузере |
| Плейлисты | Создание и управление (`POST /api/audio/playlist`) |
| Мини-плеер | В шапке сайта; музыка не останавливается при переходе по меню (SPA) |
| Метаданные эфира | `GET /api/audio/metadata?url=…` — название и исполнитель с Icecast-потоков |

Миграция: `sqlcore/sql_0010.sql`.

### Общие радиостанции (для всех)

Администратор добавляет станции, которые видят **все** пользователи в блоке «Радио сайта» на `/music` (в дополнение к личным потокам).

| URL | Описание |
|-----|----------|
| `/admin?type=RadioStations` | Управление общими станциями |
| `POST /api/admin/radio/create` | Добавить станцию (название, URL, порядок) |
| `GET /api/admin/radio/delete?id=` | Удалить станцию |

Миграция: `sqlcore/sql_0011.sql`.

Пример для production:

- **Название:** `Fetbuk Radio 320`
- **URL:** `https://radio.fetbuk.ru/ptrc320`

### SPA и фото

- `routing.js` — навигация без полной перезагрузки (для авторизованных).
- `GET /api/photo/move` — перелистывание фото стрелками.
- `ng_asset()` — сброс кэша JS/CSS по git-hash.

## Обновление с 1.6

```bash
cd /var/www/nativegallery   # или /mnt/win/nativegallery
git fetch --tags
git checkout release-1.7    # или git pull origin main
composer install --no-dev

# миграции музыки (если ещё не применяли):
mysql -u USER -p DATABASE < sqlcore/sql_0010.sql
mysql -u USER -p DATABASE < sqlcore/sql_0011.sql

# опционально — страницы с датой правки (если пропустили 1.6+):
mysql -u USER -p DATABASE < sqlcore/sql_0009.sql

sudo chown -R www-data:www-data storage uploads cdn logs
sudo chmod -R 775 storage uploads cdn logs

sudo systemctl reload nginx php8.3-fpm
```

Обновите страницу в браузере с полным сбросом кэша (Ctrl+F5), чтобы подтянулись новые JS/CSS.

## Nginx

**Правки конфигурации nginx для релиза 1.7 не требуются**, если сайт уже работает через front controller (`deploy/nginx/nativegallery.conf`). Потоки вроде `radio.fetbuk.ru` воспроизводятся напрямую из браузера.

## Конфигурация (`ngallery.yaml`)

```yaml
audio:
  upload:
    allow: true
    maxsize: 52428800
  streams:
    allow: true
```

См. [configuration.md](../configuration.md#музыка-audio).
