# NativeGallery 1.4

**Дата:** 2026-07-02  
**Тег:** `release-1.4`  
**Репозиторий:** [Lanmikeman/nativegallery](https://github.com/Lanmikeman/nativegallery)

Форк для развёртывания на Ubuntu 24.04 + Nginx + PHP 8.3 + MariaDB с функционалом, близким к [transphoto.org](https://transphoto.org).

## Основное

- Страница обновлений: `/update`, `/update?time=24`, архив по датам, фильтры
- Поиск (для авторизованных): фото, ТС, комментарии, авторы
- Публичные разделы: `/news`, `/news2`, `/misc`, `/links`
- Заявки на изменение БД с опциональной привязкой фото (`sql_0007`)
- Редактирование фото (`/lk/editimage`, API `/api/photo/edit`)
- Фотоконкурсы: cron, этапы голосования, админ-завершение
- Документация: `docs/configuration.md`, `docs/deployment.md`, `docs/routes.md`

## Обновление с 1.3

```bash
git fetch --tags
git checkout release-1.4   # или git pull && git checkout main
composer install --no-dev

mysql -u USER -p DATABASE < sqlcore/sql_0005.sql
mysql -u USER -p DATABASE < sqlcore/sql_0006.sql
mysql -u USER -p DATABASE < sqlcore/sql_0007.sql

mkdir -p storage/locks logs
chown -R www-data:www-data storage logs
chmod -R 775 storage logs

# timezone в ngallery.yaml:
# timezone: 'Europe/Moscow'

sudo NG_WEB_ROOT=/path/to/nativegallery bash deploy/setup-cron.sh
```
